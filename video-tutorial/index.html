Video Tutorial

RL Audio + RL Video Downloader

Uma vitrine em HTML para apresentar o tutorial do projeto com foco em conversão de áudio, download de vídeos, automação com Python e integração com ferramentas de mídia.

Python 3.12+ CustomTkinter FFmpeg + FFprobe yt-dlp MoviePy
O que o vídeo mostra Demonstração da interface, fluxo de conversão de mídia, corte de áudio, download de vídeos por URL e extração de MP3.
O que foi construído Duas aplicações desktop em Python com separação entre UI, validações, regras de negócio e integração com ferramentas externas.
Objetivo do projeto Evoluir de scripts simples para uma base mais próxima de um produto real, com foco em arquitetura, usabilidade e processamento de mídia.

Stack utilizada no projeto

CustomTkinter Interface desktop moderna para organizar a experiência de uso da aplicação.
FFmpeg e FFprobe Conversão, extração, leitura de metadados e operações com mídia.
yt-dlp Download de conteúdo por URL com suporte ao fluxo de extração de áudio.
MoviePy Complemento para manipulação de mídia dentro da base em Python.
threading Execução em background para manter a interface responsiva durante o processamento.
